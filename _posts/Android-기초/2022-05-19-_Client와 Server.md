# Client와 Server
- 클라이언트/서버는 두 개의 컴퓨터 프로그램 사이에 이루어지는 역할 관계를 나타내는 것
- 클라이언트는
    - 다른 프로그램에게 서비스를 요청하는 프로그램
- 서버
    - 그 요청에 대해 응답을 해주는 프로그램
